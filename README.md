## Getting Started

### Prerequisites

* npm

  ```sh
  npm install -g yarn
  ```

### Installation

1. Get a free API Key at [https://newsapi.org/](https://newsapi.org/)

2. Clone the repo

   ```sh
   git clone https://github.com/wxin9monash/NewsLens.git
   ```
3. Install required packages

   ```sh
   npm install
   ```
   
4. Start the application

   ```js
   npx expo start
   ```
